Mac users — important: macOS will block ToneField on first launch with a security warning. To fix this:

1. Open Terminal (Applications → Utilities)
2. Paste this command and press Enter:
xattr -cr /Applications/ToneField.app
3. Double-click ToneField — it will open normally.

You only need to do this once. The app is not damaged — this is a standard macOS security step for apps distributed outside the App Store.
